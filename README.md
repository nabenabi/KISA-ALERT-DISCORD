# KISA-ALERT-DISCORD

## 📌 프로젝트 소개

최근 피싱, 악성코드, 개인정보 유출 등 다양한 정보보안 위협이 지속적으로 증가하고 있습니다.  
이러한 보안 이슈는 개인뿐만 아니라 기업, 학교, 커뮤니티 운영에도 직접적인 영향을 줄 수 있습니다.

`KISA-ALERT-DISCORD`는 KISA에서 제공하는 보안 공지 RSS를 자동으로 수집하여,  
새로운 보안 위협 정보가 올라오면 Discord 채널로 빠르게 전달해주는 알림 봇입니다.

이를 통해 사용자는 KISA 홈페이지를 직접 확인하지 않아도  
최신 보안 공지와 취약점 정보를 Discord에서 편리하게 확인할 수 있습니다.

---

## 🚀 주요 기능

- KISA RSS 피드 자동 수집
- 새로운 보안 공지 확인
- Discord 채널로 알림 전송
- 중복 알림 방지
- 제목, 게시일, 링크 정보 제공

---

## 🧩 RSS란?

RSS(Really Simple Syndication)는 웹사이트의 새로운 콘텐츠를  
사용자가 직접 방문하지 않아도 구독 형태로 받아볼 수 있게 해주는 기술입니다.

KISA에서는 보안공지, 취약점 정보, 보안뉴스 등을 RSS 형태로 제공하고 있으며,  
이 프로젝트는 해당 RSS 데이터를 활용하여 Discord에 자동으로 알림을 전송합니다.

---

## 💡 개발 배경

정보보안 위협은 빠르게 확산되는 경우가 많기 때문에  
관련 정보를 얼마나 빠르게 확인하느냐가 중요합니다.

하지만 매번 KISA 홈페이지에 접속해 새로운 공지가 있는지 확인하는 것은 번거롭습니다.  
그래서 Discord를 사용하는 커뮤니티나 팀에서 보안 정보를 쉽게 확인할 수 있도록  
자동 알림 봇을 제작하게 되었습니다.
